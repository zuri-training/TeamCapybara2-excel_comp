TEAM CAPYBARA2 - EXCEL_COMP

Project Brief : A platform that helps users compare the contents of Excel files and perform actions on the result based on the user choice.



OVERVIEW


Use excel_comp to compare two excel sheets or files to see the differences between them.



FEATURES


  1. User (Unauthenticated) :

    • Can visit the platform to view basic information about it
    • View and Interact with the documentation
    • Register to view more details (no access until registered)

  2. User (authenticated) :

    • Full access to the platform
    • Allow upload csv/excel file
    • Users should get the option to :

      • Highlight duplicates in a single file
      • Highlight duplicates and return 2 files
      • Remove duplicates and return a single file
      • Remove duplicates and return 2 files
    • Show usage example to users
    • Allow user save data and come back to download
